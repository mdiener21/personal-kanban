# Labels for Boards (Kanban, Personal, Creative)

Labels go along with a [board](/boards) to help filter and find tasks.
You should use columns to manage the `state` getting to done and labels for `context` and filtering.

## For a Project Management (Kanban-style) Board

These labels are often used for categorization or prioritization, as seen in tools like Trello.

### Types of tasks

- **Bug**
- **Feature Request**
- **Research**
- **Meeting Notes**
- **Follow Up**

---
