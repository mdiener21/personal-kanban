Quick Start

  • Open controls: Click the ellipsis (‚ãÆ) button in the header.
  • Local-first: Everything runs in your browser; data is stored in localStorage.

Tasks

  • Add: Click the plus icon in a column header.
  • Edit: Click a task card to edit details (title, description, priority, due date, labels).
  • Move: Drag and drop between columns.

Columns

  • Add: Use Add Column to create a new workflow stage.
  • Reorder: Drag the column grip handle.
  • Edit/Delete: Use the column ellipsis menu.

Labels & Label Groups

  • Manage: Use Manage Labels to create, edit, delete.
  • Groups: Optionally assign labels to a group (e.g., People, Status, Activity) to keep things organized.
  • Search: Label search matches both label names and group names.

Settings

  • Open Settings from the controls menu to customize what shows on cards (e.g., priority, due dates) and your preferences.
